El proyecto se trata de un archivo csv que funciona como base de datos, en donde esta cargada la informacion de todos los pilotos, equipos, posiciones, puntos. 
Se lee el archivos csv y se almacena en una matriz.
Las funciones que tienen es: Buscar por piloto y mostrar los podios, Mostrar los datos de un equipo en una temporada especifica,
mostrar la mayor remontada de un piloto en una carrera, mostrar todos los equipos del archivo, mostrar todos los datos del archivo y
mostrar el piloto mas consistente del archivo.
